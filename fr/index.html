Comment télécharger l'application sur Android?

  • Parcourez les options de mobile et allez dans la section "Paramètres" ou "Réglages"
  • Cliquez sur "Réglages avancés" (variable selon votre Smartphone)
  • Cliquez sur "Sécurité" (variable selon votre Smartphone)
  • Activez "Sources inconnues"
  • Confirmez votre choix en cliquant sur "OK"
  • Surfez sur l'adresse Web "www.meridianbet.be"
  • Vous verrez l'écran "Téléchargez notre application Android". Cliquez sur "Android"
  • Si vous ne voyez pas cet écran: allez au "Menu" > en bas > "Téléchargez applications" > en bas > "Télécharger Android"
  • Cliquez sur "Téléchargez"
  • Parcourez vos applications téléchargées
  • Cliquez sur "app-be-release.apk"
  • Installez ensuite votre application Meridianbet.be sur votre mobile
  • Félicitations, l'application est maintenant installée avec succès sur votre mobile. Vous pouvez maintenant vous inscrire ou vous connecter.
